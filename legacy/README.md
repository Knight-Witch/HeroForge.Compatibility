Planned areas:


Rules:

- Never edit imported originals.
- Preserve original filenames where practical.
- Record author, version, source, import date, and provenance where known.
- Keep `.options.json` and `.storage.json` exports separate from source code.
- Create maintained reconstructions outside `/legacy/`.
