# Investigations

Use this directory for unresolved HeroForge behavior, unknown legacy patches, runtime probes, and compatibility questions.

Each investigation should clearly label observations as:

- confirmed,
- supported inference,
- unproven hypothesis.

Close or update an investigation with one of:

- confirmed,
- disproven,
- still unknown,
- no longer relevant.

Do not allow speculative investigation notes to silently become architecture truth.
