# Script Audits

Forensic audits of immutable legacy scripts.

Each audit should record:

- original filename,
- author/version/provenance,
- actual feature list,
- HeroForge dependencies,
- globals and runtime overrides,
- bundles intercepted,
- exact/regex patching strategies,
- cross-script dependencies,
- dead/unfinished code,
- load-order or initialization hazards,
- security/trust concerns,
- known failure modes,
- recommended feature split,
- recommended disposition.

Initial audit notes from the pre-repository bulk ZIP review will be normalized here after the original files are imported into `/legacy/`.
